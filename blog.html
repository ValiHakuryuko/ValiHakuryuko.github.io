Guide: Hosting a Local Website with XAMPP and LocalToNet

July 22, 2025

This guide helps you set up a local web server using XAMPP and make it publicly accessible using Localtonet.

Requirements

  • A PC with Windows, macOS, or Linux
  • XAMPP installed
  • Localtonet account

Step 1: Install & Configure XAMPP

  1. Download and install XAMPP from apachefriends.org.
  2. Open the XAMPP Control Panel.
  3. Start the Apache module.
  4. Test it by visiting http://localhost in your browser.

Place your files in C:\xampp\htdocs\. Then access them via http://localhost/index.html or index.php.

Step 2: Expose Localhost using Localtonet

  1. Sign up at localtonet.com
  2. Download the Localtonet application
  3. Add its path to your systemâ€™s environment variables
  4. Open terminal and run: localtonet
  5. Enter the AuthToken (found in the Localtonet dashboard)
  6. Once connected, go to "My Tokens" > "HTTP" tab and create a new tunnel
  7. Start the tunnel to generate a public URL

Now anyone with the URL can access your local project from the internet!
